# Slime Rancher API

A GraphQL and RESTful API for the Slime Rancher series. Contains endpoints for data on the slimes, foods, locations, and toys present in Slime Rancher 1 and 2.

## Features

- Both GraphQL and REST endpoints
- Data for Slime Rancher 1 and 2
- Type-safe schemas
